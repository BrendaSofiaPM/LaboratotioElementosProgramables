Arduino Electrónica

- Resumen -

Dentro de esta práctica se utilizó un placa electrónica arduino como programador de la placa Attiny85. Se hicieron las configuracions como en la aplicación de Arduino así como en la placa para con las conexiones se deben conectar lospines para que se puede programar al Attiny.

Por otra parte, es importante mencionar que una vez progrmado el Attiny ahora si se pueden realizar las acciones del programa con las conexiones necesarias de los circuitos que necesite.

- Introducción -

Arduino es una plataforma de desarrollo que se puede descargar gratuitamente, además de contar también con placas electrónicas del mismo nombre y que cuenta con un hardware libre que es reprogramable. A la placa se le pueden conectar jumpers para poder establecer conexiones con diferentes tipos de sensores (Arduino, 2014). Otra vanteja que existe de la placa es que se puede dar la indicaión de trabajar como un programador, es decir, indicar al Arduino como ISP para grabar en otra placa como lo es en un Attiny 85 la programación que se desea.

La interfaz SPI en la placa Arduino UNO utiliza los pines 11, 12, 13 y 5V para poder conectar la placa Attiny85.A la placa Arduino también se le debe indicar el boceto específico para poder indicar que funionará únicamente coo programados y se encuentra en Ejemplos > 11. ArduinoISP > ArduinoISP (Ardino, 2023).

"El Attiny85, un pequeño microcontrolador, con el que podremos realizar pequeños proyectos, en los que no necesitemos gran cantidad de memoria ni de entradas o salidas.La programación de este micro, la podremos realizar con nuestro entorno de escritorio de Arduino."(Todomicro,2023).

- Materiales -

Placa electrónica Arduino, un protoboard, cables, Attiny 85, 3 resistencias de 220 Ohms, 2 resistencias de 1kOhm, 3 leds, 2 botones, y computadora con el programa.

- Desarrollo -

Entrndo en la liga del curso de arduino básico se llevaron al protoboard los circuitos que se encontraban en la siguiente liga:https://www.tinkercad.com/classrooms/kKxYmpPDJXs/activities/8e6rkk3tuvu

Se analizaron los códigos para comprender qué el lo que ocurría y poder prototiparlo en físico para proceder a cargar el programa desde el Arduino hacia el Attiny.

Simulación

Prende los leds alternadamente.

6_Attiny 3 LEDs Delay

7_Attiny Digital Input

Prende el led cuando se presiona el botón.

8_Attiny Digital Input IF-ELSE

Prende el led cuando se presiona el botón.

9_Attiny Digital Input IF variables

Prende el led cuando se presiona el botón.

10_Attiny Digital Input OR

Prende el led con cualquiera de los dos, hace la función OR.

11_Attiny Digital Input AND

Prende cuando ambos botones están conectados.

12_Attiny Contador + 3 LEDs

Se va prendiedo un led al sumar a la variable cada que se aprieta el botón.

13_Attiny Contador + - 3 LEDs

Construcción

Para la construcción de los ciruitos se revisó los elementos que se necesitan para poder llevar a físico las simulaciones. Se fueron revisando los códigos de programación y posteriormente se subieron através del Arduino hacia el Attiny 85 para comprobar que funcionaran.

Un punto clave de esta ´ractica fue seguir los pasos necesarios pra configurar la plataforma de sofware del Ardino, para posteriormente indicar a la placa Arduino que se comportaría como un programador.

- Resultados -

Al momento de comparar las simulaciones con los circuitos en físico, se puede ver que en raelidad cada uno de ellos es idéntico. El uíco inconveniente que hbo fue en la simulación de la función AND, ya que en el simulador no se pueden presionar ambos botones al mismo tiempo.

- Conclusiones -

Con esta práctica aprendí que la placa electrónica de Arduino no solo funciona como un elemento con el que lo progrmas y realiza las funciones, si no que también puede funjir como programador y con ello tener circuitos mucho más pequeños para poder trabajar.

Otro aspecto importante es que para la simulación debemos tomar en cuenta que solo existe un puntero de mause, por lo que si querems ver si funciona correctamente un circuito con condicionales AND, será imposible presionar dos cosas al mismo tiempo y solo se podrá comprobar de manera física.
