Práctica 5. Pantalla OLED

Brenda Sofía Pastrana Monzón
Ing. Biomédica
Fecha: Septiembre 28, 2023
Arduino Electrónica

- Resumen -

Se presenta una pantalla OLED en dónde muestra ON si se presiona un botón y OFF si no está presionado.

- Introducción -

Las pantallas OLED permiten controlar pixeles para poder formar texto o gráficos, comparadas con las pantallas LCD, no necesitan retroiluminación haciendo que sean de mejor calidad y mayor contraste. OLED por sus siglas en inglés significa Organic Light-Emitting Diode que traducido sería diodo orgánico de emisión de luz. Para poder usarla es necesario contar con las librerías necesarias.(Lozano, 2020).

- Materiales -

Placa electrónica Arduino UNO, Attiny85, un protoboard, 1 resistencia de 220 Ohms, 1 botón, cables y computadora con el programa.

- Desarrollo -

Para esta práctica se utilizó de base el siguiente link: https://wokwi.com/projects/344317868853166675 en dónde se remplaza el sensor que tiene con el botón.

Se editó el código de la misma página y se realizaron las conecciones necesarias entre el Attiny85, el botón y la pantalla OLED.

Simulación

Pantalla OLED

El circuito contiene una Pnatalla OLED que al presionar el borón aparece el texto"ON" y cuando no se presiona muestra "OFF".

Construcción

En esta práctica solo se ralizó la conexión nesasaria entre el Attiny85, la pantalla OLD y un botón.

- Resultados -

Al momento de querer cambiar el programa de ejeplo con uno que funcionara con un botón fue cuestión de prueba y error hasta que se obtuvo el resultado deseado.

La pantalla OLED pone el texto "ON" cuando el botón está encendido, de lo contrario pone el texto "OFF".

- Conclusiones -

Es importante conocer el alcance que llega a tener una pnatalla OLED y que se pude imprimir casi cualquier cosa enn ella, así como se debe conectar correctamente para que su funcionamiento logre verse de manera adecuada ya que si no se limpia la pantalla se llega a ver lo del anterior texto.
