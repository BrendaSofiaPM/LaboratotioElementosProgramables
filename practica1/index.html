Arduino Electrónica

- Resumen -

Dentro de esta práctica se utilizó un placa electrónica arduino, así como el programa del mismo, con ayuda de tutoriales en ThinkerCad se relizó un repaso básico para poder comprender y recordar diferentes instrucciones que se tienen en este elemento.

Por otra parte, este curso de Arduino básico conta de dos partes, en esta práctiac sólo se realizpo la primera parte.

- Introducción -

Arduino es una plataforma de desarrollo que se puede descargar gratuitamente, además de contar también con placas electrónicas del mismo nombre y que cuenta con un hardware libre que es reprogramable. A la placa se le pueden conectar jumpers para poder establecer conexiones con diferentes tipos de sensores (Arduino, 2014).

La ventaja de usar Arduino es que puedes personalizar y agregar librerías así como mejorar el diseño de hardware, además de tener un lenguaje de programación de fácil comprensión que está basado en C++.

- Materiales -

Placa electrónica Arduino, un display de 7 segmentos, un protoboard, 4 leds, 2 botones, cables y computadora con el programa.

- Desarrollo -

Entrndo en la liga del curso de arduino básico se llevaron al protoboard los circuitos que se encontraban en la siguiente liga: https://www.tinkercad.com/joinclass/THFNAX84SDIQ.

Se analizaron los códigos para comprender qué el lo que ocurría y poder prototiparlo en el Arduino físico para proceder a cargar el programa.

Construcción

Para la construcción de estos circuitos priero se revisó la simulación de los archivos descargables, se analizaban los progrmas para posteriormente construir en el protoboard el circuito. Se centinuó poniendo el código de la simulación el programa Arduino para posteriormente cargarlo a la tarjeta electrónica Arduino.

- Resultados -
