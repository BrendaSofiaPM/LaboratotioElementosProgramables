Práctica 2. Arduino Básico II

Brenda Sofía Pastrana Monzón
Ing. Biomédica
Fecha: Agosto 31, 2023
Arduino Electrónica

- Resumen -

Se trabajó con servomotores y potenciometros para poder aprender que puedes convetir las entradas de un potenciómetro en salidas representadas por servotores.

Se comprobaba que las simulaciones realizaran lo mismo con el Arduino en físico y los componentes.

- Materiales -

Placa electrónica Arduino, 2 servomotores, un protoboard, 2 leds, 2 resistencias de 220 Ohms, 2 potenciómetros, cables y computadora con el programa.

- Desarrollo -

Para la elaboración de esta práctica, se entró a la liga https://www.tinkercad.com/classrooms/b5s2eOYhgMn/activities/7h8cznM37L1.

Se revisaban los códigos y los elementos que se necesitaban para armar el ciurcuito en físico y comprobar que era lo mismo de la simulación así con en físico.

Simulación

1_servo I

El servomotor se mueve de 0 a 90 grados, de 90 a 180 grados y se repite.

2_servo II

El sevomotor se mueve de un lado al otro de manera constante.

3_potenciometro_serial print

Conforme se mueve el potenciometro, se ve las lecturas que tiene la ventana del serial monitor.

4_servos

Ambos servomotores se mueven al mismo sentido que va girando el potenciómertro.

5_servos invertidos

Ambos servomotores se mueven pero en sentidos opuestos entre ellos conforme va girando el potenciómertro.

6_servos independientes

Se mueve un servomotor al mismo sentido que un potenciómetro, y el otro servomotor se mueve con el otro potenciómetro.

7_servos independientes y leds

El servomotor izquierdo se mueve con el potenciómetro izquiero y cuando llegan a un extremo prende el led rojo, o del otro extremo prende el led verde.

8_servo_fuente

Mueve el servomotor con una fuente externa.

Construcción

Para la construcción de los ciruitos se revisó los elementos que se necesitan para poder llevar a físico las simulaciones. Se fueron revisando los códigos de programación y posteriormente se subieron al Arduino para comprobar que funcionaran.

- Resultados -

Al momento de comparar las simulaciones con los circuitos en físico, se puede ver que en raelidad cada uno de ellos es idéntico.

El servomotor se mueve de 0 a 90 grados, de 90 a 180 grados y se repite.

El sevomotor se mueve de un lado al otro de manera constante.

Conforme se mueve el potenciometro, se ve las lecturas que tiene la ventana del serial monitor.
